# 05 — Profiles (Lookbook + Business)

## Influencer Lookbook (`/influencer/[id]`)
- [ ] Header: avatar, display name, handle, location, niche tags, verified badge
- [ ] Video pitch player
- [ ] Stat block: followers, avg views, avg likes, avg comments, engagement rate
- [ ] Reach / impressions / profile views (24h)
- [ ] Audience demographics: age-bucket bars, gender split, top locations
- [ ] Rate card (if provided)
- [ ] "Send pitch" CTA (business view)

## Business profile (`/business/[id]`)
- [ ] Header: name, handle, bio, follower count, hiring badge
- [ ] Vibe-check grid (last 6 posts)
- [ ] Target demographic summary
- [ ] Attached pitch/brief (if any)
